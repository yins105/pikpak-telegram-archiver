## PikPak 電報檔案保存支援哪些格式？

PikPak Bot 支援 Telegram 能傳輸的所有格式，基本上就是：

- **影片**：MP4、MKV、AVI 等主流格式，存好直接在 PikPak 裡串流
- **圖片**：JPG、PNG、GIF 等
- **壓縮檔**：ZIP、RAR、7Z
- **文件**：PDF、Word、Excel 等
- **音樂**：MP3、FLAC 等

單一檔案上限跟 Telegram 本身一致，約 2GB。如果是超大壓縮包分割成多個檔，每個都可以個別轉發。

另外除了電報的原生檔案，Bot 也支援你直接貼上其他平台的連結——比如 TikTok 影片連結、X (Twitter) 的影片連結——Bot 會去抓取影片內容並保存到你的 PikPak。

## PikPak 電報檔案保存的進階使用技巧

### 技巧一：善用磁力連結

除了直接轉發 TG 頻道的檔案，你也可以直接把磁力連結（Magnet Link）複製後貼給 Bot，它會在雲端伺服器幫你下載這個 BT 資源然後存進雲端。

對於那些在 BT 資源站找到的稀有影片，用這個方法非常好用——PikPak 的伺服器速度通常比家用網路快得多，遇到熱門資源基本上是「秒存」。

### 技巧二：指定儲存資料夾

在轉發給 Bot 的時候，可以附帶指令指定要存到哪個資料夾，讓你的 PikPak 空間保持整齊，不會全部亂丟在根目錄。

### 技巧三：批次轉發

在 Telegram 選多個訊息同時轉發，Bot 會一個一個處理，不用你手動逐一操作。

### 技巧四：先存再看

遇到大型影片不急著馬上看？先轉給 Bot 存好，等你有時間再開 PikPak 直接串流，不用占用手機空間。
